# Лабораторная работа №2
## Задание 1. Консольные приложения

**Исполнитель:** Мещенко Валерия, гр. 10
**Вариант:** 32 (Подсчет счастливых билетов)

### Overview
Программа предназначена для вычисления количества «счастливых» шестизначных билетов в заданном диапазоне. Реализована на языке C с разделением на логические модули.

### Usage
Для запуска в Repl.it используйте кнопку **Run**. Программа скомпилирует исходные коды из папки `src` и выведет результат в консоль.

### Building
Сборка осуществляется при помощи GCC. Основные файлы проекта:
* `src/main.c` — точка входа;
* `src/lucky.c` — реализация алгоритма;
* `src/lucky.h` — заголовочный файл.
